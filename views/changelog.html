    Version 1.1.2 (2018/08/01)

    • Fixed Slack login error happened on non-developer accounts. Updated callback function.

    Version 1.1.1 (2018/07/30)

    • Fixed bugs in login and cookie handling process.
    • Updated How to Use page.

    • Fixed mobile view. Added responsive layout.
    • Fixed logout process.
    • Fixed CAN'T Display Facebook Posts after request new data.
    • Improved fetch response time by moving HTML formating from server to client side.
    • Updated rending of Facebook embedded posts with responsive design.

    • Improved rending of Facebook embedded posts (Fix post height). Changed from using Iframe method to native JavaScript SDK.
